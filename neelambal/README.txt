
-- SUMMARY --

NEELAMBAL is a simple theme for drupal 6 based on 960 grid system. 

The different regions in this themes are:
- Header
- Top menu (Primary links)
- Sidebar left
- Main content
- Content bottom left
- Content bottom center
- Content bottom right
- Footer

--------------------------------------------------------------------------
-- REQUIREMENTS --
 --
--------------------------------------------------------------------------
-- Drupal Version --

* drupal 6

--------------------------------------------------------------------------
Browser compatibility:

* Firefox 3.5
* Google chrome
* IE 9 (because some styles contains CSS3)

--------------------------------------------------------------------------
-- Installation instructions --

* The installation process is same as usual, 
see http://drupal.org/node/456 for further information.
